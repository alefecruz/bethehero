## :arrow_forward: Clonando o repositório

```sh
$ git clone https://github.com/alefecrz/bethehero.git
```

## :wrench: Rodando backend

```sh
$ cd backend
$ yarn or npm install
```

## :package: Rodando frontend web

```sh
$ cd frontend
$ yarn or npm install
```

## :iphone: Rodando frontend mobile

```sh
$ cd mobile
$ yarn or npm install
```
